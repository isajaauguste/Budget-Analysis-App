# Budget Analysis Application
TECHIN Summer project

`Git bash` terminale suteikiam skriptam paleidimo privilegijas:  
`chmod +x ./scripts/*`  

Paleidžia skriptus vietoj papildomo rašymo  
`./scripts/check.sh`   
`./scripts/run.sh`   
Mes vengiam rašyti rankom:  
`poetry run alembic upgrade head`  
`poetry run uvicorn main:app --reload`
